Welcome to Pillr.

 An application dedicated to identifying and stopping harmful diseases.

It uses intelligent algorithms, cutting edge technology and, ofcourse, your feedback.

DEVELOPED ENTIRELY USING OPEN SOURCE TECHNOLOGY

Prescriptions

 You will be able to get a prescription from your doctor and fill that prescription at any pharmacy easily.

Reminders

 The Pillr app for your smartphone will be able to remind you exactly when to take your medicine and it will also tell you about crucial directions and steps to take during a course of treatment so that it will be effective.

Feedback

 Our app will let you comment about the effects a treatment has on your health. This will help doctors choose a better course of treatment for you, and help pharmaceutical companies make better drugs.

Risky Areas

 In case of an outbreak (flu, malaria etc.) our app will let you know, it will tell you which areas to avoid, and what precautions to take.

Pillr API (Application Programming Interface)

What Is An API

An api is a language for other applications to communicate with Pillr. They can receive and submit information.

Send Information

Programmers can simply use our submit-data api module to submit information to Pillr, for example the following line creates a user:

REQUEST

/api/submit_data/create_user?uname="UNAME"&email="EMAIL"

RESPONSE

Successful

Receive Information

Programmers can also receive specialized information, for example if a programmer wants to know how many new malaria cases are to be expected next month, he can use the following receive_data module:

REQUEST

/api/receive_data/?type="prediction"
&disease="malaria"&month="11"&year="2015"

RESPONSE

JSON_RESPONSE
{
disease: "malaria"
date: "01/11/2015"
expected_new_cases:"21"
}
